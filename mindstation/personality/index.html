关于MBTI

MBTI（Myers-Briggs Type Indicator，迈尔斯 - 布里格斯类型指标）是当下应用广泛的人格认知工具，由凯瑟琳・库克・布里格斯与女儿伊莎贝尔・布里格斯・迈尔斯，基于荣格心理类型理论研发而成。

它将人格划分为四大核心维度：

  • E/I 外倾 / 内倾：代表心理能量的来源，外倾型从外界互动获取能量，内倾型从独处与内心沉淀恢复能量；
  • S/N 感觉 / 直觉：代表认知偏好，感觉型关注现实细节与客观事实，直觉型看重联想灵感与未来可能；
  • T/F 思考 / 情感：代表决策倾向，思考型优先依据客观逻辑判断，情感型优先遵从内心感受与价值取向；
  • J/P 判断 / 感知：代表生活处事风格，判断型偏爱规划秩序，感知型习惯灵活随性。

四个维度自由组合，共形成 16 种人格类型，例如 INTJ（建筑师）、ENFP（活动家）、ISTJ（物流师）等。每种人格都拥有独特的思维特点、天生优势与成长方向。
